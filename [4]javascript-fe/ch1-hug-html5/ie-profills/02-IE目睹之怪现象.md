# 目睹之怪现象

- select

```html
<select></select>
改为
<select><option></option><select>
果然可以更新渲染,如果一开始为空，一些组件框架(LayUI)渲染起来可能会出错
```

- inline-block

```javascript
// 写两个CSS声名里，这样可以兼容ie7
.ads-close {
    display: inline-block;
    cursor: pointer;
}

.ads-close {
    _zoom:1;
    *display:inline;
}
```

- IE回车的一个怪异行为 IE中在input中回车相当于提交form，会从dom中找最近的button标签触发click事件
    - 不要忘记给button 加type="button"！！！
