# 性能分析

- jHiccup：提供平台中JVM暂停的日志和记录。
- JMH：JVM基准测试工具。
- JProfiler：商业分析器。
- LatencyUtils：测量和报告延迟的工具。
- VisualVM：对运行中的应用程序信息提供了可视化界面。
- YourKit Java Profiler：商业分析器。
