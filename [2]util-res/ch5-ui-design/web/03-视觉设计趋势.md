# 图形通感化解构

```shell
物体是由形状，材质，颜色，排列等维度所构成的。
若将主体物进行打碎、分解、叠加等方式进行重组解构，从逻辑上打破其固有感官印象，
由此物体将被赋予全新的含义。
```
