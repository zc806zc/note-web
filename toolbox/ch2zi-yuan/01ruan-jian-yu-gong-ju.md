        {
        "cmd": ["node", "$file"],
        "file_regex": "^[ ]*File \"(...*?)\", line ([0-9]*)",
        "selector": "source.javascript"
        }
        ```
		*  文件保存为 JavaScript.sublime-build
		*  重启，勾选Tools --> Build System --> JavaScript
		*  Ctrl + b执行JavaScript 程序.
```
