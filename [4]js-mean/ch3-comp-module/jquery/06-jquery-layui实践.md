# Layui

- 优点

  - 因为layer组件开始关注, 弹框组件非常好
  - 表格组件整体的设计极其优雅
  - 几乎自成体系
  - 社区

- 组件设计缺陷与建议

  - 表格行内编辑还不支持填充数据与动态更新
  - 表格不完全支持resize, f12以后分页会消失(估计已解决)
  - 弹框不支持滚动条，目前提供的参数是 是否屏蔽浏览器滚动条
  - 表格组件数据格式封装过于死板
  - 自成一家的模块化系统有点尴尬
  - 模板渲染引擎也有点尴尬
  - 校验规则变来变去的...
