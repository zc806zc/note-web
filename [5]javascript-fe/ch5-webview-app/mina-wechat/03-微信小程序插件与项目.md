# wepy与mpvue

- WePY 默认支持 Redux，在脚手架生成项目的时候可以内置
- Mpvue 是 Vue 的移植版本
- 选择mpvue的原因
  - webpack的成熟社区
  - mpvue学习成本更低
  - mpvue 基于AST，未来可能还可以支持支付宝小程序和快应用
