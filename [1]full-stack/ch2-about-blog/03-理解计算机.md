```javascript
// 1.数据以文本形式保存

// 2.B树
// 二叉查找树
每个节点最多只有两个子树
左子树都为小于父节点的值，右子树都为大于父节点的值
在n个节点中找到目标值，一般只需要log(n)次比较

B树是对二叉查找树的改进。
它的设计思想是，将相关数据尽量集中在一起，
以便一次读取多个数据，减少硬盘操作次数

// 3.索引
按照"主键"查找数据
C-ISAM库和D-ISAM库

// 4.高级功能
SQL解析器
```
