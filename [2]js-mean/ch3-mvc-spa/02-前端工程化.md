```javascript
require.ensure([], (require) => {
   callback(null, [
     require('./src/pages/profitList').default,
   ]);
 });

如果一个 es6 模块只有一个 default 输出，
那么在转换成 commonjs 的时候也一起赋值给 module.exports
即整个导出对象被赋值了 default 所对应的值
```
