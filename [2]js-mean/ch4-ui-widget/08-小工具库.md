# 布局

```javascript
// 浮动
.fl { float: left; }
.fr { float: right; }
.clearfix:after { display: block; clear: both; content: ""; visibility: hidden; height: 0; }
.clearfix { zoom: 1; }

.field-group {
    label {
      width: 100px;
      text-align: right;
    }

    input {
      border: 1px solid #ddd;
    }
}

.btn-group {
  margin-top: 10px;
}


<div class="field-group clearfix">
  <label class="fl"></label>
  <input class="fl" type="text" name="" value="">
</div>

</div>
```
