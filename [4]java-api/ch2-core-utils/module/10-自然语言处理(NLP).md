# 自然语言处理

- Apache OpenNLP：处理类似分词等常见任务的工具。
- CoreNLP：斯坦佛CoreNLP提供了一组基础工具，可以处理类似标签、实体名识别和情感分析这样的任务。
- LingPipe：一组可以处理各种任务的工具集，支持POS标签、情感分析等。
- Mallet：统计学自然语言处理、文档分类、聚类、主题建模等。


# 分词器
